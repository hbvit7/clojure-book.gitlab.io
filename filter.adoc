=== filter

[source, clojure]
----
(filter odd? (range 1 11))
----

Output

----
(1 3 5 7 9)
----

[source, clojure]
----
(filter even? (range 1 11))
----

Output

----
(2 4 6 8 10)
----

[source, clojure]
----
(filter #(> % 5) (range 1 11))
----

Output

----
(6 7 8 9 10)
----

[source, clojure]
----
(filter #(> 5 %) (range 1 11))
----

Output

----
(1 2 3 4)
----

[source, clojure]
----
(filter (fn [x] (> x 5)) (range 1 11))
----

Output

----
(6 7 8 9 10)
----

[source, clojure]
----
(defn greater-than-5 [x]
  (> x 5))
----

Output

----
#'user/greater-than-5
----

[source, clojure]
----
(filter greater-than-5 (range 1 11))
----

Output

----
(6 7 8 9 10)
