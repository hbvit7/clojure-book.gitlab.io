== Destructuring

[source, clojure]
----
;; array_destructuring.clj

(def people ["Rehmaan" "Kalaam"])

(let [[musician scientist] people]
  (println "Musician is" musician)
  (println "Scientist is" scientist))
----

Output

----
Musician is Rehmaan
Scientist is Kalaam
nil
----
